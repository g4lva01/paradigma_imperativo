Origem do Paradigma

O paradigma imperativo surgiu na década de 1940, quando os primeiros computadores digitais eletrônicos estavam sendo desenvolvidos. Esses computadores eram projetados com base na arquitetura de von Neumann, um modelo proposto por John von Neumann em 1945. A arquitetura de von Neumann é baseada em um conceito simples: um computador deve ter uma unidade central de processamento (CPU), memória para armazenar tanto dados quanto instruções, e uma unidade de entrada e saída. Nesse modelo, o fluxo de execução é sequencial, e a CPU executa instruções uma de cada vez, alterando o estado da memória ao longo do tempo. A programação imperativa é um paradigma de programação que descreve a computação como uma sequência de comandos que mudam o estado de um programa.


Esse paradigma é frequentemente contrastado com a programação declarativa, que foca no que o programa deve realizar sem especificar como alcançar o resultado. Linguagens como Ada, ALGOL, Basic, C, PHP, Java, COBOL, Fortran, Pascal, Python, Lua e Mathematica seguem o modelo imperativo. O conceito fundamental da programação imperativa é a Máquina de Turing, uma abstração matemática que representa funções computáveis. John von Neumann aprimorou esse conceito, criando uma arquitetura de computadores e essa arquitetura é a base dos computadores modernos.

Exemplo de Códico:

​#include – Inclui a biblioteca stdio.h, que permite o uso de funções de entrada e saída, como printf e scanf.

int main() – Inicia a função principal do programa.

float num1, num2, media; – Declara três variáveis do tipo float para armazenar os dois números e a média.

printf("Digite dois números: "); – Exibe uma mensagem pedindo que o usuário insira dois números.

scanf("%f %f", &num1, &num2); – Lê os dois números digitados e os armazena nas variáveis num1 e num2.

media = (num1 + num2) / 2; – Calcula a média dos dois números. printf("A média é: %.2f\n",

(media); – Exibe a média calculada, formatada com duas casas decimais. return 0; – Indica que o programa terminou com sucesso.
